# Developing MVVM JavaSciprt Aplications with Vue.js [WIP]

## 关于

这是一本关于 MVVM 库 **Vue.js** 的开源书籍。本书将会以 `v1.0.0` 版本为基础，讲述如何使用 **Vue.js** 构建 MVVM 前端项目。希望通过本书，让读者能够充分了解**Vue.js** 这一个轻量、精悍的前端 JavaScript 库。
